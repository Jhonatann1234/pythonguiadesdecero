Objetivo del Curso

Aprende Python desde cero aunque nunca hayas programado en tu vida. En pocas horas estarás creando tus propios programas y entendiendo cómo funciona la tecnología que usas todos los días.

Pasos para Empezar

  1. Instala Python → Ve a python.org y descarga gratis
  2. Escribe tu primer programa
    print("¡Hola mundo, soy programador!")
  3. Crea variables → Guarda nombres, números, listas, diccionarios...
  4. Toma decisiones con if/else → Tu programa piensa como tú
  5. Repite tareas con bucles for y while → Automatización fácil
  6. Crea tus propias funciones → ¡Código reutilizable y profesional!

Glosario Básico

Variable: donde guardas información

Función: bloque de código reutilizable

Bucle: repetir acciones automáticamente

Librería: superpoderes listos para usar
